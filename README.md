# Carla Britto

Sou bacharel em Biologia pela Universidade Federal de Alagoas e, ao longo da graduação, participei de experiências acadêmicas ligadas à ecologia quantitativa, ornitologia e monitoria em Biologia e Taxonomia de Criptógamos. Com o tempo, comecei a migrar da ciência para áreas mais criativas, explorando design, identidade visual e web design. Atualmente estudo HTML e CSS enquanto desenvolvo projetos visuais inspirados em estéticas nostálgicas da internet dos anos 2000, interfaces experimentais e identidades com personalidade própria.
